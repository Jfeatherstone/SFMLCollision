38 /*
39 We can assume that the polygon class will be used for a variety of different objects,
40 some of which will require more accurate and representative shapes than others. Because of
41 this, we will give the user the option to specifiy whether or not to simplify the polygon by
42 a certain degree on its creation.
43 
44 Note: This option is only present and important in the creation of polygons through a texture.
45 If instead a vector of verticies is provided, this enum is irrelevant, which is why our class
46 doesn't have a member variable to store the level of detail, since it is only used in the
47 constructor.
48 */
49 
50 enum class Detail {Less, More, Optimal, Exact};
